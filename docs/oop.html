Object Oriented Programming

What is OOP?

Object-oriented Programming is a programming paradigm in Python that utilises objects and classes. It aims to incorporate real-world entities such as inheritance, polymorphisms, encapsulation, and so on into programming. The main idea behind OOPs is to bind the data and the functions that work on it as a single unit so that no other part of the code can access it.

The four main principles of Object-Oriented Programming (OOP's) are:-

    Encapsulation
    Polymorphism
    Abstraction
    Inheritance
Let's try to understand how Encapsulation works,

In programming, encapsulation is the process of combining elements to form a new entity for the purpose of covering or protecting information. Encapsulation is a feature of object design in object-oriented programming.
